# Modelo 3D da Terra

Coloque aqui o arquivo `earth.glb` antes de compilar o app.

## Como obter o modelo gratuitamente

1. Acesse: https://sketchfab.com/3d-models/earth-7efa2546b8174ac398accd0c30c29b04
2. Baixe no formato `.glb`
3. Renomeie para `earth.glb`
4. Cole nesta pasta: `assets/models/earth.glb`

## Requisitos do modelo
- Formato: `.glb` (recomendado) ou `.gltf` + texturas na mesma pasta
- Tamanho recomendado: escala ~0.2 metros no app
- O modelo deve ter rotação no eixo Y para a animação funcionar corretamente
